<Section icon={Target} title="Resolución de acertijos de la Deep Web">

Los acertijos en **Welcome to the Game 3 Tanner** son significativamente más complejos que en las entregas anteriores. Rara vez se encuentra información en una sola página. En su lugar, el juego utiliza "Cadenas de pistas de páginas vinculadas", donde un nombre de usuario en un foro puede llevar a un directorio oculto en un sitio de mercado, que luego contiene un fragmento de una contraseña.

<Steps>
  <Step num="1" title="Identificar el anclaje">
    Busca un identificador único, como un alias inusual (por ejemplo, "Tanner_Watcher") o un número de serie específico en una página de inicio.
  </Step>
  <Step num="2" title="Referencia cruzada">
    Usa la herramienta de búsqueda del juego para encontrar otras páginas que contengan ese anclaje. Registra cada número o símbolo asociado que encuentres en el código fuente o en las imágenes.
  </Step>
  <Step num="3" title="Ensamblar fragmentos">
    Combina las letras o dígitos recopilados. Presta mucha atención al orden, que a menudo está dictado por marcas de tiempo o ID de página.
  </Step>
  <Step num="4" title="Ejecutar la desencriptación">
    Introduce la cadena ensamblada en la terminal de seguridad. Prepárate para un pico de red que puede atraer atención no deseada.
  </Step>
</Steps>

<Callout type="info" title="Información">
Muchos acertijos utilizan pistas visuales ocultas en el fondo de las imágenes. Pasa siempre el cursor sobre los elementos sospechosos para ver si el nombre del archivo o los metadatos revelan una cadena oculta.
</Callout>

| Elemento del acertijo | Función | Ubicación común |
| :--- | :--- | :--- |
| **Nombre de usuario** | Vincula diferentes sitios entre sí | Publicaciones de foros, perfiles de vendedores |
| **Símbolo** | Indica el orden de un código | Pies de página, imágenes ocultas |
| **Marca de tiempo** | Ayuda a secuenciar datos fragmentados | Registros de chat, registros de archivos |
| **Código hexadecimal** | Entrada directa para herramientas de seguridad | Código fuente CSS, metadatos de imágenes |

</Section>
