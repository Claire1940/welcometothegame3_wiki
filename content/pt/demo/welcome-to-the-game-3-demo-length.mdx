<Section icon={Target} title="Como Concluir o Teste (Trial Run)">

Para concluir com sucesso a demo de Welcome to the Game 3, você deve seguir as instruções fornecidas por Ronald. O ciclo principal envolve encontrar hashes, que são longas sequências de caracteres alfanuméricos ocultas na Darknet.

Encontrar essas hashes é um processo de várias etapas. Algumas estão **escondidas à vista de todos** em uma página da web, muitas vezes disfarçadas como parte do texto. Outras estão **escondidas dentro de elementos**, exigindo que você clique em palavras ou imagens específicas para revelá-las. As hashes mais difíceis estão escondidas no **código-fonte da página da web**, que você deve inspecionar manualmente.

| Tarefa | Descrição | Recompensa |
| :--- | :--- | :--- |
| **Encontrar a Hash 1** | Localizada na Wiki inicial ou em links secundários | Progresso para a Decodificação |
| **Encontrar a Hash 2** | Frequentemente exige visualizar o código-fonte ou clicar em imagens | Componente Final da Chave |
| **Contratar Agente** | Pagar Doscoin a um agente via Crypt Chat | Fragmento de Chave Decifrado |
| **Compilar a Chave** | Combinar os fragmentos de acordo com seus índices | Conclusão do Teste |
