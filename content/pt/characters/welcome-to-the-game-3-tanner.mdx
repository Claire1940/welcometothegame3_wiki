<Section icon={Target} title="Resolvendo Quebra-cabeças da Deep Web">

Os puzzles em **welcome to the game 3 tanner** são significativamente mais complexos do que nas entradas anteriores. A informação raramente é encontrada em uma única página. Em vez disso, o jogo usa "Correntes de Pistas de Páginas Interligadas", onde um nome de usuário em um fórum pode levar a um diretório oculto em um site de mercado, que então contém um fragmento de uma senha.

<Steps>
  <Step num="1" title="Identificar a Âncora">
    Encontre um identificador único, como um pseudônimo incomum (ex: "Tanner_Watcher") ou um número de série específico em uma página inicial.
  </Step>
  <Step num="2" title="Referência Cruzada">
    Use a ferramenta de busca do jogo para encontrar outras páginas que contenham essa âncora. Registre cada número ou símbolo associado encontrado no código-fonte ou nas imagens.
  </Step>
  <Step num="3" title="Montar Fragmentos">
    Combine as letras ou dígitos coletados. Preste muita atenção à ordem — muitas vezes ditada por carimbos de data/hora ou IDs de página.
  </Step>
  <Step num="4" title="Executar Decriptação">
    Insira a sequência montada no terminal de segurança. Esteja preparado para um pico de rede que pode atrair atenção indesejada.
  </Step>
</Steps>

<Callout type="info" title="Informação">
Muitos quebra-cabeças utilizam pistas visuais escondidas no fundo das imagens. Sempre passe o cursor sobre elementos suspeitos para ver se o nome do arquivo ou os metadados revelam uma sequência oculta.
</Callout>

| Elemento do Puzzle | Função | Localização Comum |
| :--- | :--- | :--- |
| **Nome de Usuário** | Vincula diferentes sites | Postagens em fóruns, perfis de vendedores |
| **Símbolo** | Indica a ordem de um código | Rodapés de página, imagens ocultas |
| **Carimbo de Data/Hora** | Ajuda a sequenciar dados fragmentados | Logs de chat, registros de arquivo |
| **Código Hex** | Entrada direta para ferramentas de segurança | Fonte CSS, metadados de imagem |

</Section>
